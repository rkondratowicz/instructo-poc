# NPM Package Management Instructions

This instruction set provides guidance for working with NPM packages.

## Key Principles

- Always check package security before installation
- Use `npm audit` regularly to identify vulnerabilities
- Prefer pinned versions for production dependencies

## Common Commands

- `npm install <package>` - Install a package
- `npm update <package>` - Update a package
- `npm audit` - Check for security vulnerabilities
- `npm audit fix` - Automatically fix security issues
